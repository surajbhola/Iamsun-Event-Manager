# Iamsun Events Manager App

Events Manager is a user-friendly app designed to streamline event organization. Developed with Tanstack Query and backed by Node.js, this app allows users to effortlessly add, edit, and delete events. With a focus on simplicity, users can provide event details like name, image, description, date, time, and location. The app's intuitive interface ensures smooth navigation, complemented by a quick search feature for finding specific events. The Node.js backend guarantees reliable and efficient performance, making event management a breeze for users.
